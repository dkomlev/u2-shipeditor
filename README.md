# U2 Ship Editor v0.6

Static build. No deps.

Quick start:
1) Unzip.
2) Open index.html directly, or serve locally (python -m http.server 8080).
3) Pick size/type, click Apply Archetype to autofill.
4) Export JSON.

Folders:
- index.html
- js/: schema, presets, nominals, validator, migrate, app
- config/: shipconfig.schema.json
- configs/: example JSONs
